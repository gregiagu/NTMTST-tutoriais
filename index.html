Tutoriais para ajudar nas aulas

Como muita gente está com dificuldade em coisas básicas de informática, decidimos criar alguns tutoriais para ajudar os alunos a acompanharem as aulas.

Basta seguir os passos descritos aqui para praticar em casa e treinar para a hora da aula. Aliás, praticar é a palavra de ordem aqui: pratiquem, pratiquem e pratiquem! Ainda mais porque nas aulas nós usamos computadores com o sistema operacional Ubuntu, que é um pouco diferente do sistema operacional Windows, que a maioria de nós está acontumado.

Mas por que usar Ubuntu? A resposta é fácil: Ubuntu é um sistema operacional livre e gratuíto. Ninguém paga nada para usa-lo. Basta baixar e instalar no computador. Diferente do Windows, que é um sistema operacional de código fechado e é vendido pelo empresa Microsoft.

Como abrir o Visual Studio Code

Para abrir o Visual Studio Code, também conhecido como VSCode, basta clicar no ícone na barra lateral do ubuntu, conforme a figura abaixo.

Com o VSCode já aberto, com o mouse, aponte para o menu Arquivo e clique.

No próximo menu que abrir, selecione a opção Novo Arquivo de Texto e um novo arquivo será aberto.

Insira o código JavaScript aqui. Conforme a figura acima.

Repare que surgiu uma bolinha branca onde mostra a figura. Essa bolinha mostra que o arquivo não está salvo. Agora vamos salvar esse arquivo.

Abra novamente o menu Arquivo e desta vez clique em Salvar.

A primeira linha do arquivo será considerada como nome do arquivo. Mude esse nome. Escreva um nome bem bonito para o seu arquivo. Ah! Não esqueça de terminar com .js.

Escolha uma pasta onde será salvo o arquivo e clique em Save.

Pronto. O Arquivo está salvo. Repar que agora não tem mais bolinha branca.

Rodando código com o Node JS

É fácil rodar código JavaScript usando no NodeJS. Eu mostro como.

Primeiro passo: abra qualquer arquivo com código JavaScript.

Costuma terminar com .js no final do nome do arquivo.

Clique no ícone acima. Fica na barra lateral direita.

Agora, repare que abriu uma nova coluna de conteúdo a direita do aplicativo? Clique no botão Executar e Depurar. É o primeiro botão de baixo para cima.

Após clicar no botão de Executar e Depurar, aparecerá uma lista de opções no topo da janela. Escolha a opção Node.js. Geralmente é a primeira opção.

Agora é só ver o resultado do teu código no terminal que abrir abaixo da janela.
